Componente Motor Java


El componente Motor Java es una utilidad de la capa lógica que permite extender las funcionalidades Java de Coloso. A través de este componente el desarrollador puede tener acceso a:

  • Cargar clases, intefaces, atributos, operaciones y paquetes con sus diferentes modificadores, a partir de los modelos visuales, directamente a las interfaces de programación y compilación Java.
  • Generar código a partir de los elementos Clase e Interfaz en los diferentes diagramas. Para acceder a esta funcionalidad.
  • Ejecutar los plugins para extener la funcionalidad de Coloso
