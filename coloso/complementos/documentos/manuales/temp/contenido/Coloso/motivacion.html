Motivación de Uso
 
Coloso ofrece una amplia variedad de herramientas para diferentes fases en el proceso de creación de software, teniendo siempre en cuenta una motivación central: facilidad de uso.

A diferencia de otras herramientas disponibles para realizar tareas similares, Coloso mantiene un entorno simple, intuitivo y agradable de usar. Adicionalmente, desde el punto de vista ingenieril,  los diferentes frameworks y componentes ofrecidos permiten permiten flexibilidad al momento de incorporar nuevas herramientas a Coloso.

Los diferentes componentes y plugins de Coloso pueden ser licenciados de manera independiente o como un paquete completo, para adaptarse a las necesidades particulares de cada cliente. Por ejemplo se puede pasar de un paquete de software orientado a diseño mediante UML, hasta una herramienta completa para diseño, modelamiento, desarrollo y pruebas de software.
