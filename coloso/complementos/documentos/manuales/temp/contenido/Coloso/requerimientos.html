Requerimientos

 

Hardware Mínimo

Para ejecutar y utilizar el ambiente de desarrollo ingenieril Coloso se recomienda como mínimo la siguiente configuración:

  • Procesador Pentium IV de 2.0 Ghz o superior.
  • 512 mb de memoria.
  • 200 mb espacio en disco duro.
  • Java(TM) Runtime Enviroiment 6.0 o superior.

Software

Coloso esta creado con la tecnología Java(TM) de Sun Microsystems®, por lo que puede correr bajo plataformas que soporten la máquina virtual de Java, mas específicamente Java(TM) Plataform, Standard Edition Runtime Enviroiment Version 6 o superior.

Coloso viene con una versión portable de la plataforma para las diferentes versiones de Windows, en caso de otro sistema operativos las respectivas versiones de la máquina virtual para Windows®, Solaris®, Linux® y Apple OsX®  se pueden descargar desde :

  http://www.java.com/es/download/manual.jsp



Para el caso de los sistemas de la familia Windows, la máquina virtual de Java se encuentra incluida en el paquete de Coloso, por lo que no es necesaria su instalación previa.

Para desarrollar aplicaciones Java mediante alguno de los frameworks de programación de Coloso, es necesario instalar el Java Development Kit (JDK) version 6 de Sun Microsystems®. El JDK se puede descargar desde:

http://java.sun.com/javase/downloads/index.jsp
