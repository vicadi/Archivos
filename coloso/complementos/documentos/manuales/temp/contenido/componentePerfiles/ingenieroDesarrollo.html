Perfil Ingeniero de Desarrollo

 

Este perfil está ligado a la impelementación y mantenimiento de soluciones de software. Para el caso Coloso este perfil se ve expresado a través de las herramientas de desarrollo IAE y los diagramas de marcos.

Este componente permite insertar un perfil en las clases, como se observa en la Figura 1.

  • Clase Hoja: Indica una clase que no puede ser extendida o tener subclases. Esto se puede realizar bien por mecanismos provistos por el lenguaje Orientado a Objetos o por convención. Este patrón es aplicable a todos los diagramas en los que se puedan crear clases.
